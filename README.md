# FoFM Exam Q&A

A closed-book revision deck for **Fundamentals of Foundation Models**.

This fork keeps the original Astro + Vue exam-practice framework and replaces the content with FoFM-specific chapters, questions, sources, and review metadata.

## Content

- W1-W6: LLMs, transformers, data, tokenizers, scaling laws, SFT, RLHF, and reasoning.
- W7-W8: distributed computing, DDP/FSDP, ZeRO, and FlashAttention.
- W9-W11: vision foundation models, autoencoders, diffusion, CLIP-style VLMs, and VAEs.

Sources currently mapped:

- `SS24` Summer 2024 final
- `SS25` Summer 2025 final
- `WS26` Winter 2026 retake

## Commands

```bash
npm install
npm run build
npm run dev
```

FoFM content is generated by:

```bash
python tools/generate_fofm_content.py
```

For personal revision. Answers are condensed from the available FoFM exam papers; `AI-generated` cards are marked.
