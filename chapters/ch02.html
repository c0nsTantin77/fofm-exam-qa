2.1 Neural Networks & Fully-Connected Layers

A neural network is a composition of functions $y=h(g(f(x)))$. A fully-connected (affine / linear) layer computes $Y = XW + b$ with $X\in\mathbb{R}^{N\times D}$, $W\in\mathbb{R}^{D\times M}$, $b\in\mathbb{R}^{1\times M}$. FC layers see all inputs (global features), need a fixed input size, and are linear — two stacked affine layers collapse into one, so non-linear activations are required between them.

2.1.1Open🔥 3×Summary II.7 Q2d
Given two affine layers $y_1 = XW_1 + B_1$ and $y_2 = y_1 W_2 + B_2$, show they can be written as a single affine layer. What does this imply?
Answer

$y_2 = (XW_1+B_1)W_2 + B_2 = X(W_1 W_2) + (B_1 W_2 + B_2)$, which is again of the form $XW' + b'$ with $W' = W_1W_2$ and $b' = B_1W_2+B_2$.

Implication: stacking linear layers without a non-linearity between them gives no extra expressive power — you need non-linear activation functions to build deep, expressive networks.

💡 Extended memory

This is *the* justification for activation functions. Without them a 100-layer net = one linear layer. Composition of linear maps is linear.

Can two consecutive dropout layers with keep/drop probabilities $q$ and $p$ be replaced by one dropout operation? Explain.
Answer

Yes. Neurons are zeroed independently in each layer, so a neuron survives only if it survives both. The combined effect is a single dropout — e.g. with drop-probabilities $p,q$ the effective drop probability is $p+q-pq$ (survival $=(1-p)(1-q)$).

2.2 Backpropagation

Backprop applies the chain rule from the loss backwards. For a FC layer $Y=XW+b$ with upstream gradient $\text{dout}=\frac{\partial L}{\partial Y}$: $\frac{\partial L}{\partial X}=\text{dout}\,W^\top$, $\frac{\partial L}{\partial W}=X^\top\text{dout}$, $\frac{\partial L}{\partial b}=\sum_n \text{dout}$. Because the loss is a scalar $L:\mathbb{R}^{N}\times\mathbb{R}^{M}\to\mathbb{R}$ we only ever need gradients of scalars, never full Jacobians.

2.2.13Open1×Summary II.8 Q4a
Which property of loss functions lets us perform backpropagation without computing full Jacobians?
Answer

The loss is a scalar function $L:\mathbb{R}^{N}\times\mathbb{R}^{M}\to\mathbb{R}$, and all variables in a network are ultimately scalars assembled into matrices/tensors. We only ever need $\frac{\partial L}{\partial v}$ for scalar $v$, so gradients have the same shape as the variables — no Jacobian matrices required.

💡 Extended memory

A vector→vector layer would need a Jacobian, but a scalar loss collapses everything to gradients (one number per parameter), making backprop cheap.

(Toy network, Leaky ReLU $\alpha=0.5$) Backpropagation problems ask you to do a forward pass then update a weight via SGD. What is the derivative of Leaky ReLU used in the backward pass?
Answer

Leaky ReLU $f(x)=\max(\alpha x, x)$ has derivative $f'(x)=1$ for $x>0$ and $f'(x)=\alpha$ for $x<0$ (here $\alpha=0.5$). In backprop you multiply the upstream gradient by this local derivative element-wise, depending on the sign of each pre-activation from the forward pass.

2.4 Softmax & Numerical Stability

Softmax turns logits into a probability distribution: $\text{Softmax}(x)_i = \dfrac{e^{x_i}}{\sum_j e^{x_j}}$. It is shift-invariant: subtracting $\max(x)$ from every logit leaves the output unchanged but prevents overflow of $e^{x_i}$.

2.4.1Open2×Summary II.7 Q3f
Softmax can suffer numerical instability. Show that subtracting $\max(x)$ from every logit does not change the output: $\text{Softmax}(x-\max(x))_i = \text{Softmax}(x)_i$.
Answer

$\text{Softmax}(x-\max(x))_i = \dfrac{e^{x_i-\max(x)}}{\sum_j e^{x_j-\max(x)}} = \dfrac{e^{x_i}e^{-\max(x)}}{e^{-\max(x)}\sum_j e^{x_j}} = \dfrac{e^{x_i}}{\sum_j e^{x_j}} = \text{Softmax}(x)_i.$ The common factor $e^{-\max(x)}$ cancels.

💡 Extended memory

Subtracting the max makes the largest exponent $e^0=1$, so no overflow — the standard 'log-sum-exp' stabilization trick, output identical.

Explain why minimizing the cross-entropy $H(p,q)$ between class probabilities $q$ (network) and target $p$ is equivalent to minimizing the KL divergence $D_{KL}(p\|q)=H(p,q)-H(p)$.
Answer

Because $H(p)$ — the entropy of the fixed target distribution $p$ — does not depend on the model parameters; it is a constant with zero gradient. So $\min_\theta D_{KL}(p\|q)=\min_\theta\big(H(p,q)-H(p)\big)=\min_\theta H(p,q)$.

💡 Extended memory

Dropping a parameter-independent constant doesn't change the argmin — that's why CE training = KL minimization.

2.6 Regression Losses (MSE & MAE)

MSE / L2 $\frac{1}{N}\sum(\hat y-y)^2$ — smooth, differentiable, but sensitive to outliers (squares large errors).
MAE / L1 $\frac{1}{N}\sum|\hat y-y|$ — robust to outliers, but not differentiable at 0 (handled by sub-gradient). $N$ = total number of residuals (over batch × all output dims).

2.6.1Open2×Summary II.8 Q4c
In depth estimation most pixels have very small loss but a few have very large loss. Out of [MAE, MSE, BCE, CE], which loss is better and why?
Answer

MAE (L1) is better: it is less sensitive to outliers than MSE (which squares the large errors), so it fits the majority of small-residual pixels better instead of being dominated by a few huge residuals.

💡 Extended memory

MSE punishes outliers quadratically → a few bad pixels dominate. MAE weights all residuals linearly → robust. (BCE/CE are classification losses, irrelevant here.)
